# 终极网盘与聊天系统 🚀

这是一个基于 Streamlit 构建的综合平台，具有以下功能：

## 功能特性

### 📂 网盘 & 文件管理器
- **私人文件**: 注册用户的安全个人存储空间。
- **公共文件夹**: 访客用户的共享空间。
- **高级功能**:
  - **搜索**: 递归搜索文件名。
  - **预览**: 直接在浏览器中查看图片、播放音频/视频。
  - **Zip 下载**: 将整个文件夹打包下载为 Zip 压缩包。
  - **全面管理**: 创建文件夹、批量上传文件、删除文件。
- **性能优化**:
  - **缓存**: 优化文件列表和配置加载，体验更流畅。
  - **后台任务**: 自动清理任务在后台线程运行，避免阻塞。

### 📦 寄件箱
- **临时存储**: 发送带有留言的文件。
- **安全访问**: 生成唯一的取件码以提取包裹。
- **自动过期**: 设置保留天数（1-30天），过期后文件自动删除。

### 💬 聊天系统
- **私信 (新!)**: 注册用户之间的一对一私人对话，支持文件分享。
- **公共聊天**: 开放的匿名聊天室，支持文件附件。
- **群组聊天**:
  - 创建带有密码的私人小组。
  - 设置群组过期时间和消息保留策略。
  - 搜索消息并按日期筛选。
  - 转让所有权和管理群组设置。

### ⚙️ 管理员控制台
- **用户管理**: 重置密码、删除用户、切换管理员状态。
- **系统配置**: 设置群组创建限制、最大生命周期和消息保留时间。
- **用户概览**: 查看注册用户及其活动。

## 安装指南

1. 安装依赖:
   ```bash
   pip install -r requirements.txt
   ```

2. 运行应用程序:
   ```bash
   streamlit run app.py
   ```

## 默认账号

默认管理员账号从 `.streamlit/secrets.toml` 配置文件中获取。请修改该文件中的用户名和密码。

首次运行后，管理员账号会自动创建到用户数据库中。

## 优化说明

本项目使用 `st.cache_data` 和后台线程 (`ThreadPoolExecutor`) 来确保 UI 在处理大文件或执行清理任务时保持响应。
